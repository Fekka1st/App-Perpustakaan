## Requirements

-   PHP >= 8.0 (or higher)
-   Composer
-   Node.js
-   NPM
-   MySQL

## Installation

1.  Clone Repository
2.  Run `composer install`
3.  Run `npm install`
4.  Run `npm run dev`
5.  Buat Database lalu update .env bisa juga cp .env.example .env
6.  Run `php artisan migrate --seed`
7.  Run `php artisan serve`
8.  Login dengan user dibawah
    -   Email: `admin@mail.com`
    -   Password: `12345`
9.  Enjoy!\_
